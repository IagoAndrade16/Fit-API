# Provest API

A back-end for a older project that I make a few years ago

## Main Stacks
- Node JS
- TypeScript
- TypeORM
- Postgres
- Docker

## Routes

- POST → /courses

- POST → /users

- POST → /users/auth

- GET → /users/profile

- GET → /courses

- PATCH → /users

- PATCH → /courses

- DELETE → /courses
